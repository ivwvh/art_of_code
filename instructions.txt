Инструкции по запуску приложения:

1. Создайте новую папку
2. Разверните внутри папки виртуально окружени Python:
    В командной строке Windows
    2.1 cd <имя_папки>
    2.2 python -m venv <название_виртуального_окружения>
3. Активируйте виртуальное окружение:
    В командной строке Windows
    3.1 <название_виртуального_окружения>\Scripts\activate.bat
4.  Установите необходимые зависимости
    В командной строке Windows
    4.1 pip install Django
5.  Создайте проект Django и перейдите в его папку:
    5.1 django-admin startproject <имя_проекта>
    5.2 cd <имя_проекта>
6. Скопируйте приложение в папку проекта
7. Замените файлы настроек:
    7.1 /etc/settings.py ->  /<имя_проекта>/<имя_проекта>/settings.py
    7.2 /etc/urls.py -> /<имя_проекта>/<имя_проекта>/urls.py
8. Осуществите миграцию БД:
    8.1. python manage.py makemigrations
    8.2 python manage.py migrate
9. Запустите приложение:
    9.2 python manage.py runserver
